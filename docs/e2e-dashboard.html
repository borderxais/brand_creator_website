npm Commands

npm run e2e:upBoot stack, migrate, seed. Idempotent.
npm run e2e:exploreBoot only. Stack stays up for manual testing.
npm run e2e:resetTruncate DB and re-seed.
npm run e2e:downStop + remove containers and volumes.
npm run e2e:agentRun Playwright agent suite. Writes .e2e/runs/latest/summary.md.
npm run e2e:debug -- <runId>Print summary + last 50 lines of api/web logs.
npm run e2eLegacy fast path: dev webServer, no compose. Smoke only.
